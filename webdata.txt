                        CORE DESCRIPTION
================================================================================

Free online image and PDF tools with privacy-first browser-based processing.
No uploads, no registration. 100% private, secure, and free forever.
All processing happens locally in the user's browser — files never leave
the user's device.

                      TOOLS — IMAGE (8 TOOLS)
================================================================================

1. Resize Image        /image-tools/resize
   Resize photos by pixels or percentage — for social media, web, and print

2. Crop Image          /image-tools/crop
   Crop images to any aspect ratio — Instagram, YouTube thumbnails, profiles

3. Rotate Image        /image-tools/rotate
   Rotate images 90°, 180°, 270° — fix orientation, correct EXIF data

4. Compress Image      /image-tools/compress
   Reduce image file size by up to 80% without quality loss — web & email

5. Convert Format      /image-tools/convert
   Convert images between JPEG, PNG, WebP, and GIF formats

6. Add Watermark       /image-tools/watermark
   Add custom text watermarks — adjustable font, size, opacity, position

7. Flip Image          /image-tools/flip
   Flip images horizontally or vertically — mirror effects, selfie fix

8. Grayscale           /image-tools/grayscale
   Convert to black & white, sepia, or inverted colors — artistic effects
